# SoapSwirl Studio

SoapSwirl Studio is a native SwiftUI iOS app for sketching a soap swirl, choosing scent mood colors, previewing loaf/cut-face visuals, and saving local soap swirl sketch drafts.

- App language: English (United States) / en-US
- Storage: local JSON only; no login, backend, tracking, or cloud sync
- Premium: local visual/theme packs only; base sketch creation remains free
- App Intent: quick capture a soap swirl draft and keep manual creation available

Open `SoapSwirlStudio.xcodeproj` in Xcode and run the `SoapSwirlStudio` scheme on an iOS simulator.
